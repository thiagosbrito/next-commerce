## Getting Started

### Prerequisites

- Node.js 22+
- pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/next-commerce.git
   cd next-commerce
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Run the development server:
   ```bash
   pnpm dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
